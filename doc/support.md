# Workavera Support

Workavera is a self-hosted AI workspace. If you need help with the iOS app,
contact us using one of the options below.

## Contact

- Email: **wumulaozu@gmail.com**
- GitHub Issues: https://github.com/xusenlin/workavera/issues

When reporting a problem, please include:

- Workavera app version
- iOS or iPadOS version
- Device model
- Workavera server version
- Steps to reproduce the problem

Do not include passwords, API keys, authentication tokens, model credentials, or
private workspace content.

## Self-hosted server and account support

Accounts and workspace data are managed by the administrator of the Workavera
server you connect to. For account access, account deletion, backups, data
retention, or server availability, contact your server administrator.

## App Store purchases

For a missing Lifetime Supporter entitlement, open **Settings → Support
Workavera** and select **Restore Purchases** while signed in to the Apple Account
that made the purchase.

## Privacy

Read the [Workavera Privacy Policy](privacy-policy.md).

# Workavera 支持

Workavera 是一款自托管 AI 工作空间。如需 iOS App 使用支持，请通过以下方式联系我们。

## 联系方式

- 邮箱：**wumulaozu@gmail.com**
- GitHub Issues：https://github.com/xusenlin/workavera/issues

反馈问题时，请提供：

- Workavera App 版本
- iOS 或 iPadOS 版本
- 设备型号
- Workavera 服务器版本
- 问题复现步骤

请勿提交密码、API 密钥、登录令牌、模型凭据或私人工作区内容。

## 自托管服务器与账号支持

账号和工作区数据由你所连接的 Workavera 服务器管理员管理。账号访问、账号删除、备份、数据
保留或服务器可用性问题，请联系服务器管理员。

## App Store 购买支持

如果永久支持者权益没有恢复，请使用购买时的 Apple 账号登录，然后打开
**设置 → 支持 Workavera → 恢复购买**。

## 隐私

请阅读 [Workavera 隐私政策](privacy-policy.md)。
